Welcome to Auto Tracker!

Follow The Steps Below to Set Up Your Auto Tracker

1. Click the button below to copy the AutoTracker script.

Clicking the button will copy the necessary JavaScript code to your clipboard.

2. Create a new bookmark in your browser.

To add a bookmark you can right click the current page and bookmark it, then edit the URL, or in the bookmark bar below the url section, you can right click and ADD PAGE

3. Paste the copied script into the URL field of the bookmark.

In the URL field of your new bookmark, paste the JavaScript code you copied earlier.

4. Name the bookmark whatever you like and save it to your bookmarks bar.

Give your bookmark a name that you will recognize and save it.

5. Click the bookmark each time an action is made to activate the tracker.

Clicking the bookmark will activate the tracker and show the active indicator.

The tracker active indicator will inform you that the sale will be tracked.
